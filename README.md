# Sign Up Form
A [sign-up form project](https://www.theodinproject.com/lessons/node-path-intermediate-html-and-css-sign-up-form) from The Odin Project, built to practice and demonstrate proficiency in complex form design, CSS layout, and implementing a provided design with HTML and CSS.

## Features
- Clean, modern UI design matching the provided specification
- Interactive form elements with visual feedback
